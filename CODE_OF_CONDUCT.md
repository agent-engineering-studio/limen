# Code of Conduct

Limen is an open-source civic and scientific software project. We want contributors, users and maintainers to collaborate respectfully and responsibly.

## Expected behavior

- Be respectful, constructive and precise.
- Discuss evidence, methods, reproducibility and impact.
- Welcome newcomers and help them find focused contributions.
- Credit datasets, standards, scientific references and external work.
- Use cautious language when discussing risk, hazards and public safety.

## Unacceptable behavior

- Harassment, insults, personal attacks or discriminatory language.
- Publishing private information, secrets, credentials or personal data.
- Misrepresenting Limen as an official warning system without authorization.
- Making unsupported public-safety claims from model output.
- Repeated off-topic promotion or spam.

## Enforcement

Maintainers may edit, hide or remove comments, close issues, block users, or restrict participation when behavior harms the project or community.

## Scope

This code applies to project spaces including GitHub issues, pull requests, discussions, documentation and community channels connected to Limen.
